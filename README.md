# Excel For Managers

This repository contains a comprehensive collection of Microsoft Excel practical exercises covering basic to advanced concepts in data analysis, financial modeling, dashboards, and automation. It includes the use of string functions (LEN, LEFT, RIGHT, MID), lookup functions (VLOOKUP, INDEX, MATCH), SUMPRODUCT, SUMIFS, Pivot Tables, conditional formatting, and What-If analysis tools like Goal Seek and Data Tables. The files also demonstrate financial calculations such as depreciation (SLN, DB), EMI computation (PMT, PPMT), IRR/XIRR, future value (FV), tax scenarios, investment analysis, and stock market return and risk analysis. In addition, it covers advanced visualization techniques including line, column, bar, scatter, combo, thermometer, bullet, and dynamic charts with slicers and linked legends, along with complete dashboard creation methods. The repository further includes array formulas, circular references, macro creation for mathematical and financial operations, regression analysis, descriptive statistics using Data Analysis ToolPak, and automation using VBA. Overall, this project serves as a practical guide to mastering Excel for business analytics, financial analysis, reporting, and dynamic dashboard development.
